### Steps

Set up a virtualenv and activate

    virtualenv venv -p python3

    source venv/bin/activate

#### Install requirements

    pip install -r requirements.txt

    npm install

#### Run the code

    python manage.py runserver
    npm run dev

#### Behold!

Go to http://localhost:8000/
